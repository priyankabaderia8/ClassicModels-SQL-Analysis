# 🚗 Classic Models SQL Analysis

## 📌 Project Overview

This project showcases SQL-based data analysis using the **Classic Models** sample database. The database represents a fictional company that sells classic model vehicles and contains information about customers, employees, offices, products, orders, order details, payments, and product lines.

The objective of this project is to explore the dataset, answer business-related questions, and generate meaningful insights using SQL. It demonstrates practical SQL skills commonly required in data analyst roles.

---

## 🎯 Objectives

* Explore and understand the database structure.
* Analyze customer purchasing behavior.
* Identify top-selling products.
* Evaluate sales performance.
* Examine employee and office performance.
* Generate business insights through SQL queries.
* Practice advanced SQL concepts using real-world scenarios.
